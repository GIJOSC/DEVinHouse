Os componentes são utilizados para reaproveitamento de código e padronização de interface. Isso torna o React uma tecnologia muito flexível para a solução de problemas e para a construção de interfaces reutilizáveis, uma vez que cada um destes componentes pode ser manipulado de maneira distinta.

Componentes permitem você dividir a UI em partes independentes, reutilizáveis e pensar em cada parte isoladamente. Tais funções são chamadas “puras” porque elas não tentam alterar suas entradas e sempre retornam o mesmo resultado para as mesmas entradas.

Props são Somente Leitura Independente se você declarar um componente como uma função ou uma classe, ele nunca deve modificar seus próprios props.
