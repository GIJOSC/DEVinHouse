Qual o formato de um Action na arquitetura Flux? Exemplifique com uma função?

As Actions são objetos que indicam diretamente uma ação realizada no software, essa Action pode ser algo disparado pelo próprio usuário ou algo disparado pela própria aplicação. A Action possui sempre uma propriedade type única que é utilizada pelo Redux para distinguir qual ação está sendo realizada, esse conceito recebe o nome de Action Type.

Exemplo de action

{
type: 'LOGIN',
payload: {
username: 'admin',
password: '123456'
},
}
